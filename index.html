ABG Standards Editor

0.0.1
  • Add Node
  • Pan/Zoom
  • Click in the open space to add a standard
  • Drag between standards to add a relationship
  • Alt-drag a standard to move graph layout
  • Click a standard or a relationship to select it
  • When a standard is selected:
    • R toggles reflexivity
    • Delete removes the standard
  • When a relationship is selected:
    • L(eft), R(ight), B(oth) change arrows
    • Delete removes the relationship

What is the ABG Standards Editor?

The ABG Standards Editor is a tool used to visualize and edit relationships between standard elements.
